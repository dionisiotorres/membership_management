Website Membership Management

Enroll, Renew, Earn and Forecast with membership plans on your Odoo website. A structured Membership Model ensures a steady revenue stream for your business.

The module helps you to analyze and forecast the demand for services by studying current trend.

With Odoo membership management module you can offer membership on products and services on Odoo website.

It facilitates the customers to purchase and manage membership plans from the Odoo website.The module also provides options for auto-renewal and Trial Period while creating a membership plan which can also be customize per customer.

Moreover, the customers can cancel or update their current membership plans from their website account.

Basic Features
  • The admin can create and publish membership products on Odoo website.
  • Various membership plans can be created according to different available products and interested customers.
  • The customer can purchase a particular product to activate the desired membership plan.
  • The plan gets automatically activated for the customer once the order is confirmed in Odoo.
  • Alternatively, the Odoo user can manually assign a membership plan to a customer from the Odoo backend.
  • The user can also enable auto-renewal option in a plan to automatically reactivate the plan for the particular customer upon expiration.
  • A trial period can also be added while creating a membership plan in Odoo.
  • The customers can cancel and update his/her plan from the website account.
  • The invoices for the membership plans are automatically generated with 'In Progress' state.
  • Admin can choose to override the product price with membership plan price or can define it accordingly.

Activating The Membership Plans

The user can choose between two options to set the start date of the Membership plan - 'Start immediately' or 'Trial Period'

Trial Period:

This option allows you to add a Trial Period to the Membership Plan.
  • First of all, You will create a Membership Plan.
  • Go to Sales Order >> Configuration >> Membership information. Click on Membership plans.
  • Open any membership plan. Here. we have a CheckBox name 'trial period',
  • Enable the Checkbox.
  • Two new fields are introduced to set the duration of the trial.
    • Note: The trial period can also be enabled/disabled for specific customers separately while placing the membership order irrespective whether the plan contains a trial period or not.

Start immediately

Selecting this option would activate the plan immediately after the order is confirmed in Odoo. This means the plan would not have any Trial period. The billing would start immediately.

Auto- Renewal Option for Membership Plan

  • Enabling this option, automatically renews the membership for the customer when it expires to prevent disruption in the services.

Note: The auto-renewal option can also be managed separately in each order irrespective of the membership plan.

Product

Membership Products

Creating Membership Products:

  • Go to Sales >> Product.
  • Here we have a CheckBox named 'Is Membership Type'. Enable this to make the product a membership product .
  • (Optional) Create Variants and enable 'Is Membership Type' for each variant.
  • Then, choose a Membership plan for the product.

Membership Source

Type of Sources

The Type of source defines the origin of the membership order. The Odoo user can either directly create the membership for the customer. Or, The user can create a membership and associate it with a sales order.

Manual
To create a Membership with source 'Manual', we need to create Membership from Back-end on the click of Create button.
Sale Order
Create Membership with Sale Order:
  • To create a Membership with source 'Sale Order', Create a Quotation of Sale Order with at least one membership type product in the order line.
  • When we click on the 'Confirm' button on the 'Sale Quotation' then, automatically a membership is created in 'In-progress' state.
